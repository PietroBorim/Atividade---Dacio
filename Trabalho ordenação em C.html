O que é Selection Sort?

O Selection Sort (ordenação por seleção) é um algoritmo simples de ordenação. Ele funciona encontrando o menor valor de uma lista e colocando ele na posição correta.

Esse processo é repetido até que todos os elementos estejam organizados.

Como o algoritmo funciona

  1. O algoritmo procura o menor número da lista.
  2. Troca esse número com o primeiro elemento.
  3. Depois procura o menor número do restante da lista.
  4. Repete o processo até ordenar tudo.

Código do Selection Sort em C


#include <stdio.h>

void selectionSort(int v[], int n){

    int i, j, menor, temp;

    for(i = 0; i < n-1; i++){

        menor = i;

        for(j = i+1; j < n; j++){

            if(v[j] < v[menor]){
                menor = j;
            }

        }

        temp = v[i];
        v[i] = v[menor];
        v[menor] = temp;
    }

}

int main(){

    int v[5] = {5,3,8,1,2};
    int i;

    selectionSort(v,5);

    printf("Vetor ordenado:\\n");

    for(i=0;i<5;i++){
        printf("%d ", v[i]);
    }

}

Exemplo de Ordenação

Vetor inicial:

5, 3, 8, 1, 2

Depois do Selection Sort:

1, 2, 3, 5, 8
